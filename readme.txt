== Description ==
A easy plugin to import dummy data for themes by Aki Themes.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for Aki Toolkit
3. Activate Aki Toolkit plugin from your Plugins page.
4. Go to the Appearance -> Demo Data

= From WordPress.org =

1. Download aki-toolkit.
2. Upload the 'aki-toolkit' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Aki Toolkit from your Plugins page.
4. Go to the Customizer (Appearance -> Customize) to add custom color to your site.
